Comment ça marche & Sécurité

Transparence totale sur ce que nous lisons, ce que nous envoyons à l'IA, et ce que nous stockons (spoiler : rien).

Étape 1
🔗

Connexion Gmail

Vous autorisez MailSense à accéder à votre Gmail via Google OAuth2 — le même protocole utilisé par toutes les applications Google (Gmail mobile, etc.).

Nous demandons 3 permissions : lire vos emails, gérer les labels et modifier les labels d'emails. Nous ne lisons pas le corps complet, n'envoyons rien, et ne supprimons rien.
Étape 2
👁️

Ce que nous lisons

Pour chaque email, nous récupérons uniquement :

  • L'adresse expéditeur (champ "De")
  • L'objet du mail (champ "Sujet")
  • Les 200 premiers caractères du corps (extrait visible dans la liste Gmail)

Nous n'accédons pas aux pièces jointes, aux destinataires, ni au corps complet des emails.

Étape 3
🤖

L'IA de classification

Nous utilisons Claude par Anthropic pour classer vos emails dans 11 catégories. Les données (expéditeur + objet + extrait) sont envoyées à l'API Anthropic pour classification.

Anthropic applique une politique de non-stockage des données API. Vos emails ne sont pas utilisés pour entraîner des modèles. Voir la politique de confidentialité d'Anthropic.
Sécurité
🛡️

Zéro stockage — comment c'est possible ?

  • Aucune base de données (SQL, NoSQL, fichiers)
  • Votre token Gmail est stocké en RAM uniquement, durée de vie 2 heures
  • Vos emails transitent en mémoire et sont purgés après le tri
  • Nos serveurs ne conservent aucune donnée entre les sessions
  • Aucun compte utilisateur créé, aucun email enregistré
Techniquement : tout est traité via des variables en mémoire RAM. Après le tri, un gc.collect() explicite purge toutes les références. Si le serveur redémarre, tout est perdu — c'est voulu.
Labels
🏷️

Labels créés dans Gmail

MailSense crée 11 labels directement dans votre Gmail : ADMINISTRATIF, BANQUE_FINANCE, FACTURES_PAIEMENTS, CONTRATS_ABONNEMENTS, EMPLOI_PRO, SANTE, TRANSPORT_VOYAGE, NEWSLETTERS_MARKETING, RESEAUX_SOCIAUX, PERSONNEL, SPAM_PHISHING.

Si un label du même nom existe déjà, un suffixe _MS est ajouté pour éviter les conflits. Vous pouvez supprimer ces labels à tout moment depuis les paramètres Gmail.
